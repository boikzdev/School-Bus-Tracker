A real-time location tracking solution designed to solve transportation delays and communication gaps between school administration and students. 

> **Live Demo:** [View on Replit]([https://replit.com/@boikunthadev/School-Bus-Tracker])

## 📌 The Problem
In many school systems, students and parents face uncertainty regarding bus arrival times, leading to long wait times and safety concerns. This project provides a transparent, real-time dashboard to monitor transit status.

## ✨ Key Features
* **Real-time GPS Tracking:** Live location updates powered by Firebase Realtime Database.
* **Cross-Platform:** Built with Flutter for seamless performance on Web and Mobile.
* **Admin Dashboard:** Simplified interface for school coordinators to manage routes.
* **Minimalist UI:** Focuses on speed and low data consumption for areas with varying connectivity.

## 🛠️ Tech Stack
* **Frontend:** Flutter / Dart
* **Backend & Database:** Google Firebase (Realtime Database & Authentication)
* **Deployment:** Replit / Base44 (Cloud-native "Vibe Coding" workflow)

## 🚀 Getting Started

### Prerequisites
* Flutter SDK installed
* Firebase Project configured

### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/boikzdev/School-Bus-Tracker.git)
    ```
2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run the app:**
    ```bash
    flutter run
    ```
